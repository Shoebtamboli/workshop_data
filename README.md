## How to get started with the application

### Clone the repo
```
git clone https://github.com/Shoebtamboli/workshop_data.git
```
### install dependencies
```
bundle install
```
### Set up DB
```
rails db:reset
rails db:seed
```
### Start the server
```
rails s
```
